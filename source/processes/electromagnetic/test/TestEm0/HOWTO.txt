
Before doing anything else, please confirm that you are
using the correct tags. Tags used for the profiler test 
are listed in the result file (resDDMMMYY.txt).

===========================================================

 
For Performance test.

1) copy architecture.gmk to geant4beta/config

 % cp architecture.gmk ../../../../../config/.

2) for the first time, remove all of tmp and lib files,
  so that making sure all libraries are compiled in same 
  options.

 % cd ../../../../..
 % rm -rf lib tmp

3) compile everything

 % cd source
 % gmake glob

4) make test binary

 % cd processes/electromagnetic/test
 % gmake

5) run it

 % ../../../../../bin/AIX-AFS/TestEm0 prof.mac


------------------------------------------------------------

For Checking the results

1) edit TestEm0.cc

 Remove comment mark (//) in front of #define Em0NoOptimize 1

2) Re-build

 % gmake

3) run it

 % ../../../../../bin/AIX-AFS/TestEm0 prof.mac
